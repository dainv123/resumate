

## **Ý tưởng chính**

Ứng dụng giúp freelancer / người đi làm cập nhật **CV & Portfolio tự động** từ những project mới, hoặc từ một Job Description (JD).
👉 Giảm thời gian chỉnh sửa thủ công, tăng tính chuyên nghiệp, cập nhật profile theo đúng ngôn ngữ nhà tuyển dụng.

---

## **Core Features**

### 1. Import & Quản lý CV gốc

* Người dùng upload CV (PDF/Word/LinkedIn export).
* App phân tích nội dung (AI/NLP) → trích xuất **học vấn, kinh nghiệm, kỹ năng, dự án**.
* Lưu thành dạng **CV Data Model (JSON)** để dễ cập nhật.

### 2. Cập nhật CV từ Project mới

* Người dùng nhập **Project Info** (tên dự án, vai trò, tech stack, kết quả).
* AI tóm tắt, chuẩn hóa thành 1–2 bullet point phù hợp format CV.
* Tự động chèn vào CV đang lưu trữ.

### 3. Cập nhật CV từ JD (Job Description)

* Người dùng upload/link JD.
* AI phân tích **keywords, skill yêu cầu, nhiệm vụ chính**.
* Đề xuất bản CV tailored → highlight những trải nghiệm liên quan.
* Xuất bản CV phiên bản "fit JD".

### 4. Portfolio Sync

* Ngoài CV, app có thể xuất nội dung project ra **Portfolio page** (online profile template).
* Giúp freelancer showcase nhanh cho khách hàng.

### 5. Export đa định dạng

* Xuất CV: PDF, Word, hoặc **ATS-friendly format**.
* Export portfolio: share link (online hosted profile).

### 6. Smart Suggestions

* AI gợi ý cải thiện:

  * "Bạn có thể nhấn mạnh kết quả dự án này bằng số liệu."
  * "Job này yêu cầu Python, hãy thêm từ khóa vào mô tả project A."

---

## **User Flow (Simple)**

1. Đăng ký → Upload CV → app phân tích & lưu.
2. Thêm project mới → app gợi ý nội dung → update CV.
3. Upload JD → app tạo CV tailored → export.
4. Có thể tạo Portfolio link → gửi khách hàng.


---

## **Khác biệt so với LinkedIn Premium**

1. LinkedIn Premium chỉ giúp **hiển thị profile tốt hơn** và có data về recruiter view.
2. App này giúp **tự động hóa việc tạo & cập nhật CV** → thứ LinkedIn không làm.
3. Có **CV bản tailored theo JD** (LinkedIn thì chỉ có 1 profile cố định).
4. Hỗ trợ Portfolio cho freelancer (LinkedIn portfolio khá hạn chế).

## **Business Model**

* Freemium:

  * Free: import 1 CV, update tối đa 3 projects/tháng.
  * Pro: unlimited updates, JD-tailored CV, portfolio hosting, export ATS-optimized PDF.
* Mục tiêu khách hàng: Freelancer, consultant, designer/dev đổi job thường xuyên.
